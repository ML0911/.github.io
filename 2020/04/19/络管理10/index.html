发表于 2020-04-19 分类于 网络管理 阅读次数： Valine：

1.访问控制列表适用于?

C C. 所有网络,如IP、IPX

2.访问列表是路由器的一种安全策略,你决定用一个标准ip访问列表来做安全控制，以下为标准访问列表的例子为?

B B. access-list 10 deny 192.168.10.23 0.0.0.0

3.下列哪些访问列表范围符合IP范围的扩展访问控制列表?

B B. 100-199

4.锐捷交换机采用访问控制列表描述正确的是?

D D.交换机上必须采用命名方式的ACL

5.写出标准IP ACL可以进行匹配的项目

DD.以上都不是

6.当端口由于违规操作而进入"err-disabled"状态后，使用什么命令手工将其恢复为UP状态?

A A.errdisable recovery

7.在访问控制列表中地址和掩码为168.18.64.0 0.0.3.255示的IP地址范围是()?

B B.168.18.64.0~ 168.18.67.255

8.对于IP访问控制列表描述正确的是?

D D.匹配成功马上停止

9.当学生A两台Star-R2624 路由器在做路由器广域网口连通实验时误将本端路由器V.35线缆的插头插倒了。结果发现ping直连的路由器不通，请问此时查看本端路由器的状态应该是?

CC、Serial 0 is down，line protocol is down

10.输出"Encapsulation PPP,LCP OPEN OpenIPCP,CCP,CDCP,loopback not set "的说法正确的是?

D D.数据可经过此链路传输

11.FTP服务器的iP地址在192 1620.02子网内，为了保证FP服务器的安全，不允许任何用户telnet到该FTP服务器,则需要配的访问列表条目为?

D D.access-list 110 deny tcp any 192.168.20.0 0.0.0

12.以下为标准访问列表选项是?

C C、access-list 1 permit 172.168.10.198 255.255.0.0

13.R2624路由器如何验证接口的ACL应用?

DD. show acess-list

14.在网络中，为保证192.168.10.0/24网络中WWW服务器的安全,我们只允许访问WEB服务,现在采用访问控制列表来实现，正确是?

A A、access-list 100 permit tcp any 192.168.10.0 0.0.0.255 eq www

15.下列哪项不是交换机端口安全的基本功能?

BB.数据包过滤

16.下列哪项不是关于交换机端口安全的描述?

B B.端口安全功能只能在aggregate端口上进行配置

17.以下情况可以使用访问控制列表准确描述的是?

C C.禁止所有使用Telnet的用户访问我的主机

18.下列条件中，能用作标准ACL决定报文是转发或还是丢弃的匹配条件有?

A A.源主机IP

19.你刚创建了一个扩展访问列表101,现在你想把它应用到接口上,通过以下哪条命令可以把它应用到接口上?

B B.ip access-group 101 out

20.在访问列表中有一条规则如下:access-list 131 prmit ip any192.168.10.0 0.0.0.255 eq ftp在该规则中，any的意思是表示:

C C.允许所有的源地址; D.允许255.255.255.255 0.0.0.1

21.配置pap验证客户端的命令有?

A.RA(config-if)#encapsulationppp;

B.RA(configif)#pppauthenatication pap

22.关于输出"PPP: IPCP_ reqci: return CONFACK'的说法正确的有?

A A.LCP协商已成功完成

B B.路由器已同意IP参数

23.在锐捷系列交换上,端口安全的默认配置有哪些?

AA.默认为关闭端口安全

BB.最大安全地址个数是128

CC.没有安全地址

24. 配置端口安全存在哪些限制?

A.一个安全端口必须是一个 Access 端口，及连接终端设备的端口， 而非 Trunk 端口;

B.一个安全端口不能是一个聚合端口（Aggregate Port）;

C.一个安全端口不能是 SPAN 的目的端口

25.访问控制列表具有哪些作用?

A A.安全控制

B B.流量控制

C C.数据流量标识

26.IP访问列表分为哪两类?

A A.标准访问列表

D D.扩展访问列表
