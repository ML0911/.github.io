网络安全9

发表于 2020-04-15 分类于 网络安全 阅读次数： Valine：

1、简述Windows系统权限、安全标识符、注册表的概念，注册表的作用。

（1）Windows系统权限：用户权限：登录权限：账户通过身份验证前所具备的权限； 操作权限：账户通过身份验证之后所具备的权限，利用权限可以控制资源被访问的方式， User组成员对某个资源拥有“读取”操作权限、Administrators组成员拥有“读取+写入+删除”操作权限。
（2）安全标识符：在Windows系统中，系统是通过安全标识符（SID）对用户进行识别的，SID可以应用于系统内的所有用户、组、服务或计算机，因为SID是一个具有惟一性、绝对不会重复产生的数值。
（3）注册表的概念：注册表是存储计算机的配置信息的数据容器
（4）注册表的作用：在系统中，注册表是一个记录32位驱动的设置和位置的数据库。当操作系统需要存取硬件设备，它使用驱动程序，甚至设备是一个BIOS支持的设备。

2、简述Linux系统的权限模型和安全访问。（选学）

（1）权限模型： Linux系统是一个多用户多任务的分时操作系统，任何一个要使用系统资源的用户，都必须首先向系统管理员申请一个账号，然后以这个账号的身份进入系统。用户的账号一方面可以帮助系统管理员对使用系统的用户进行跟踪，并控制他们对系统资源的访问;另一方面也可以帮助用户组织文件，并为用户提供安全性保护。每个用户账号都拥有一个惟一的用户名和各自的口令。用户在登录时键入正确的用户名和口令后，就能够进入系统和自己的主目录。
（2）安全访问：xinetd是管理其他网络服务守护进程如telnet、pop3、imap等的守护进程，xinetd的主要配置文件是/ etc/ xinetd.conf,并且提供一个配置目录/ etc/xinetd.d

3、简述系统口令破解中字典破解、暴力破解概念。

（1）字典破解：字典破解指通过破解者对管理员的了解，猜测其可能使用某些信息作为密码，同时结合对密码长度的猜测，利用工具来生成密码破解字典。
（2）暴力破解：暴力破解指对密码可能使用的字符和长度进行设定后，对所有可能的密码组合逐个实验。

4、简述Web服务器安全设置的内容

(1) Windows IIs 安全设置：Internet信息服务器(即Internet Information Server, 简称IIS)是 用户创建信息服务器的最重要的组件，最新的版本是Windows 2003中包含的IIS 6.0。与Window NT Server完全集成在- -起，因而能够利用Windows NT Server和NTFS(NT File System，NT的文件系统)内置的安全特性，建立强大、灵活且安全的Internet和Intranet站点。
(2) linux apache安全设置：Apache服务器是Internet中应用最为广泛的Web服务器软件之一。Apache服务器源自美国国家超级技术计算应用中心(NCSA) 的Web服务器项目，需经过精心配置后，才能使它适应高负荷，大吞吐量的互联网工作。Apache利用配置文件/etc/httpd/conf/ httpd.conf进行参数配置及其虚拟目录的管理。

5、简述数据库系统安全含义。（选学）

数据库系统安全是指为数据库系统建立安全保护措施，以保护数据库系统软件和其中的数据不因偶然和恶意的原因而遭到破坏、更改和泄漏。
